# NG Visuals V2 — GitHub Pages

## What changed
- Larger header logo with no duplicate “NG Visuals” text.
- Real portfolio photography.
- Working review carousel with arrows and slide/fade movement.
- Zero-cost assistant that works on GitHub Pages.
- Email, SMS and WhatsApp buttons configured.
- Average delivery: 1–2 days.

## Important: assistant limitation
The included assistant is a browser-based agency FAQ assistant. It works free and needs no API key.

A real ChatGPT/OpenAI assistant cannot safely run directly on GitHub Pages because an API key placed in public JavaScript would be stolen. A real model requires:
- an API account,
- a private backend/serverless function,
- and potentially usage charges.

## Important: reviews
The displayed reviews are explicitly marked as demo content. Do not remove that label or present invented reviews as genuine clients. Replace each demo with verified feedback after completing real projects.

## Update your existing GitHub repository
1. Extract this ZIP.
2. Open your `ng-visuals` repository.
3. Click **Add file → Upload files**.
4. Upload and replace:
   - `index.html`
   - `style.css`
   - `script.js`
   - `assets/logo.png`
5. Click **Commit changes**.
6. Wait 1–3 minutes.
7. Refresh the GitHub Pages site with Ctrl+F5.

GitHub Pages republishes automatically after the commit.
